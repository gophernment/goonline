## Basic syntax - Control Flow if/else

```go
    if a != b {
        println("a not equal to b")
    } else if a < b {
        println("a less than b")
    } else {
        println("ok")
    }
```

## Basic syntax - Control Flow if/else with statement

```go
    if ok := IsCorrect(); ok {

    if n, err := strconv.Atoi("5"); err != nil {
        log.Println(err)
    } else {
        log.Printf("the number is %d\n", n)
    }
```
